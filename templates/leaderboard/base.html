As of October 17, the ~6 month time period for the GREAT3 challenge has begun, with submissions due by April 30, 2014. We are now releasing all simulations for the control, multiepoch, and real_galaxy (12 branches total), with the final branches to be released soon.

We plan soon to apply an overall metric recalculation to all methods. Therefore, do not be perturbed if you see a small change in your metric score in any given branch leaderboard over the coming days. This will be an effective renomalization so that target-meeting methods score Q~1000. All changes will be made fairly across all methods. No metric changes will alter relative rankings between methods.

The variable_psf branches, nicknamed Popcorn, were previously available but are temporarily disabled: this is while we run final checks on new simulations that fix a problem in the original release. Recently, an issue has been identified in the PSFs used for the multiepoch simulations: we are regenerating these data, and will inform everyone when this is done, but are keeping the data available for now as changes are minor and the format will be entirely preserved.
